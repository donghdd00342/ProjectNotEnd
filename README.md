## Workflow
- Mỗi người tạo lấy 1 branch từ "master" sau đó code và push branch của mình lên rồi lên Github tạo một "Pull request", tạo xong thì ping anh để merge.
- Nhớ pull master về trước tiến hành "Pull request"
